# Socket.io Chat App

A real-time chat app built with Socket.io.

## Features

- Real-time chat
- Multiple chat rooms
- Usernames
- Online user list

## Getting Started

1. Clone the repo.
2. Install dependencies.
3. Start the server.
4. Open `http://localhost:3000` in your browser.

## Usage

1. Choose a username and chat room.
2. Start chatting.

## Tech Stack

- Node.js
- Express.js
- Socket.io
